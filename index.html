📉 PAGERANK — Vulnerableبيج رانك — قابل للاختراق
• Teleports to any random page in the entire web
• Can be manipulated by link farms
• Measures importance, not trustworthiness
• Vulnerable to spam networks and SEO poisoning
✅ TRUSTRANK — Hardenedتصنيف الثقة — محصّن
• Teleports only to manually verified seed pages
• Trust degrades the further from seeds
• Measures trustworthiness, not just importance
• Resistant to link farm manipulation

📐 Clustering Coefficientمعامل التجميع
Real social circles are "incestuous" — your friends know each other. Fake bot networks lack cross-links, producing a very low clustering coefficient.

C = (triangles × 2) / (deg × (deg-1))
📈 Expansion Rateمعدل التوسع
If a node connects to 100 people (First Hop) with zero mutual friends but reaches 10,000 at Second Hop — it triggers an artificial amplification flag.

Bot speed: too many friends acquired too fast.
📏 Path Lengthطول المسار
If the Source Node is only 2 hops from many unrelated groups, it indicates a coordinated campaign to spread misinformation or spam across the network.

Short paths to diverse communities = red flag.

Bubble Sort
فرز الفقاعة
خوارزمية فرز الفقاعة هي أبسط خوارزميات الفرز. تعمل عن طريق المقارنة المتكررة بين العناصر المتجاورة وتبديلها إذا كانت في الترتيب الخاطئ. تُستخدم في التعليم لفهم مبدأ الفرز، لكنها غير فعّالة للبيانات الكبيرة.
Bubble Sort repeatedly steps through the list, compares adjacent elements, and swaps them if they are in the wrong order. It is primarily used as an educational tool due to its simplicity, but is inefficient on large datasets.

def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n - i - 1):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
    return arr

Binary Search
البحث الثنائي
البحث الثنائي خوارزمية فعّالة للبحث في مصفوفة مرتبة. تعمل بتقسيم نطاق البحث إلى نصفين في كل خطوة، مما يجعلها أسرع بكثير من البحث الخطي. تُستخدم في قواعد البيانات وأنظمة الملفات.
Binary Search efficiently locates a target value in a sorted array by repeatedly halving the search interval. It is far faster than linear search and is widely used in databases, file systems, and search engines.

def binary_search(arr, target):
    low, high = 0, len(arr) - 1
    while low <= high:
        mid = (low + high) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            low = mid + 1
        else:
            high = mid - 1
    return -1

Merge Sort
فرز الدمج
فرز الدمج خوارزمية تعتمد على مبدأ "فرّق تسُد". تقسّم المصفوفة إلى نصفين، تفرز كلاً منهما بشكل مستقل، ثم تدمجهما. تتميز بأدائها الثابت O(n log n) في جميع الحالات وهي خوارزمية مستقرة.
Merge Sort divides the array in half recursively, sorts each half independently, then merges them back together. It guarantees O(n log n) performance in all cases and is a stable sorting algorithm used in production systems.

def merge_sort(arr):
    if len(arr) <= 1:
        return arr
    mid = len(arr) // 2
    left  = merge_sort(arr[:mid])
    right = merge_sort(arr[mid:])
    return merge(left, right)

def merge(left, right):
    result = []
    i = j = 0
    while i < len(left) and j < len(right):
        if left[i] <= right[j]:
            result.append(left[i]); i += 1
        else:
            result.append(right[j]); j += 1
    return result + left[i:] + right[j:]

Dijkstra's Algorithm
خوارزمية ديكسترا
خوارزمية ديكسترا تجد أقصر مسار بين عقدة المصدر وجميع العقد الأخرى في رسم بياني موزون بأوزان موجبة. تُستخدم في بروتوكولات التوجيه في الشبكات، وخرائط التنقل، وأنظمة أمن الشبكات الاجتماعية لتتبع انتشار المعلومات.
Dijkstra's Algorithm finds the shortest path from a source node to all other nodes in a weighted graph with non-negative weights. It is used in network routing protocols (OSPF), GPS navigation, and social network analysis to model information propagation.

import heapq

def dijkstra(graph, start):
    dist = {node: float('inf') for node in graph}
    dist[start] = 0
    heap = [(0, start)]
    while heap:
        d, u = heapq.heappop(heap)
        if d > dist[u]:
            continue
        for v, w in graph[u]:
            if dist[u] + w < dist[v]:
                dist[v] = dist[u] + w
                heapq.heappush(heap, (dist[v], v))
    return dist

PageRank
خوارزمية بيج رانك
خوارزمية بيج رانك (المستخدمة في محرك بحث جوجل) تقيس أهمية الصفحة بناءً على عدد وجودة الروابط الواردة إليها. تُطبَّق في تحليل الشبكات الاجتماعية لتحديد العقد الأكثر تأثيراً وكشف حسابات سيبل المزيفة.
PageRank (used by Google Search) measures a node's importance based on the quantity and quality of incoming links. In social network security, it is applied to identify influential nodes and detect Sybil accounts by revealing anomalous trust distributions.

def pagerank(graph, damping=0.85, iters=100):
    n = len(graph)
    rank = {node: 1 / n for node in graph}
    for _ in range(iters):
        new_rank = {}
        for node in graph:
            total = sum(
                rank[src] / len(graph[src])
                for src in graph
                if node in graph[src]
            )
            new_rank[node] = (1 - damping) / n + damping * total
        rank = new_rank
    return rank

Levenshtein Distance
مسافة ليفنشتاين
مسافة ليفنشتاين تقيس الحد الأدنى من عمليات التحرير (إدراج، حذف، استبدال) اللازمة لتحويل سلسلة نصية إلى أخرى. تُستخدم في كشف التصيد الاحتيالي عن طريق مقارنة أسماء النطاقات لاكتشاف النطاقات المشبوهة المشابهة للنطاقات الشرعية.
Levenshtein Distance measures the minimum number of single-character edits (insertions, deletions, substitutions) required to transform one string into another. In phishing detection, it compares domain names to identify typosquatted or lookalike domains.

def levenshtein(s, t):
    m, n = len(s), len(t)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    for i in range(m + 1):
        dp[i][0] = i
    for j in range(n + 1):
        dp[0][j] = j
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if s[i-1] == t[j-1]:
                dp[i][j] = dp[i-1][j-1]
            else:
                dp[i][j] = 1 + min(
                    dp[i-1][j],
                    dp[i][j-1],
                    dp[i-1][j-1]
                )
    return dp[m][n]
